This skill guides the process of performing web searches to find information that may not be available in the current context or when the user explicitly requests to search the web.

# Web Search Methodology
1. **Identify Search Intent**: Understand the user's query and determine the specific information they are seeking. Clarify any ambiguous terms or concepts to ensure accurate search results.
2. **Formulate Search Queries**: Create effective search queries that are specific, relevant, and likely to yield useful results. Use keywords, phrases, and operators to refine the search.
3. **Select Search Engines**: Choose appropriate search engines or platforms based on the type of information being sought (e.g., Google for general information, academic databases for research papers, social media for trends).
4. **Evaluate Search Results**: Assess the credibility, relevance, and quality of the search results. Prioritize authoritative sources, recent publications, and well-regarded websites.
5. **Extract and Summarize Information**: Extract key information from the search results and summarize it in a clear and concise manner. Provide context and explanations as needed to ensure the user understands the information.
6. **Cite Sources**: When providing information from web searches, always cite the sources to allow the user to verify the information and explore further if desired.
- Use proper formatting for citations (e.g., author, title, publication date, URL).

# Web Sources 
- **Official Documentation**: Always prioritize official documentation for accurate and reliable information.
- **Reputable Websites**: Use well-known and respected websites for general information.
- **Academic Papers**: For research-based queries, refer to peer-reviewed journals and conference papers.
- **News Outlets**: For current events, use established news organizations.
- **Community Forums**: For practical advice and troubleshooting, consider forums like Stack Overflow or Reddit.
